Spotify Clone (HTML & CSS)

Description: This project is a front-end clone of Spotify built using only HTML and CSS. It focuses on recreating Spotify’s modern UI, layout, and visual components without using JavaScript, ensuring clean structure and responsive styling.

Technologies Used: HTML5, CSS3 (Flexbox, Grid)

Features:

  • Spotify-inspired modern UI design
  • Fully responsive layout
  • Sidebar navigation and music sections
  • Playlist and album cards layout
  • Footer music player UI (static)
  • Clean and structured CSS
