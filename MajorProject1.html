Wanderlust – Airbnb Clone

Description: Wanderlust is a full-stack Airbnb clone that allows users to explore, add, and manage travel listings. The platform includes authentication, listing management, and user reviews, providing a real-world booking website experience.

Technologies Used: HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, EJS, Bootstrap

Features:

  • User authentication (Login & Signup)
  • Add, edit, and delete property listings
  • Upload images for listings
  • Review and rating system
  • Authorization & protected routes
  • Responsive design for all devices
