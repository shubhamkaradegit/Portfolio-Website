Mini Project: Search Movie (React)

Description: Search Movie is a mini React application that allows users to search for movies and view basic details using a public movie API. The project focuses on API integration, state management, and dynamic UI updates.

Technologies Used: React.js, JavaScript (ES6), HTML, CSS, Movie API

Features:

  • Search movies by title
  • Fetch real-time data from movie API
  • Dynamic rendering using React state
  • Reusable React components
  • Responsive UI
