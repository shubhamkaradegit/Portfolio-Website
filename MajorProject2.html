Major Project: Blinkit Clone

Description: Blinkit Clone is a major front-end project developed using pure HTML and CSS. The project replicates the UI of Blinkit’s quick-commerce platform, focusing on layout design, responsiveness, and pixel-perfect UI components.

Technologies Used: HTML5, CSS3 (Flexbox, Grid)

Features:

  • Blinkit-inspired homepage UI
  • Category-based product sections
  • Product cards with pricing and offers
  • Sticky navbar and search bar UI
  • Responsive design for mobile and desktop
  • Clean and scalable CSS structure
