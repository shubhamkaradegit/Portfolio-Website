Mini Project: Weather App (React)

Description: Weather App is a mini React application that provides real-time weather information for any city. The app fetches live data from a weather API and displays temperature, humidity, and weather conditions with a clean and responsive UI.

Technologies Used: React.js, JavaScript (ES6), HTML, CSS, Weather API

Features:

  • Search weather by city name
  • Real-time weather data using API
  • Temperature, humidity & weather condition display
  • React hooks (useState, useEffect)
  • Responsive UI
