Mini Project: Todo App (React)

Description: Todo App is a mini React application that helps users manage daily tasks efficiently. Users can add, delete, and mark tasks as completed, focusing on React state management and component-based architecture.

Technologies Used: React.js, JavaScript (ES6), HTML, CSS

Features:

  • Add new tasks
  • Delete tasks
  • Mark tasks as completed
  • React state & props usage
  • Reusable components
  • Responsive UI
